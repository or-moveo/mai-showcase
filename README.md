# M.ai Showcase

Landing page for **M.ai** — the AI strategy & agentic platforms arm of [Moveo Group](https://moveo.group).

## Overview

A standalone, single-file HTML showcase featuring:

- Animated hero with gradient visuals
- Stats bar with scroll-triggered counter animations
- Services grid (6 cards) — Copilots, AI Infrastructure, Workflow Automation, Smart Dashboards, AI Governance, Team Enablement
- Case studies with metrics
- AI Journey process flow
- Client logos — enterprise & startup partners

## Tech

- Single `index.html` with embedded CSS & JS
- No build tools or frameworks required
- Google Fonts: Urbanist (headings) + Assistant (body)
- Scroll animations via `IntersectionObserver`
- Fully responsive (mobile-first)

## Run

Open `index.html` in a browser — no server needed.

```sh
open index.html
```
